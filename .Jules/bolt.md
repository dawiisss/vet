## 2026-06-10 - [App Component Lazy Loading] Learning: [When optimizing React applications for performance, using `React.lazy` and `Suspense` to dynamically import heavy, conditionally rendered components (like Modals) is a highly effective way to reduce the initial bundle size and speed up the initial time-to-interactive (TTI).] Action: [Always look for modals or heavy off-screen UI elements in root-level components (like App.tsx) and convert static imports to lazy imports.]
